# Weather App (Python)

This project is a simple Python console application that shows weather information for selected cities.

No internet connection required.

## Features
- Get weather by city
- Display temperature and conditions
- Uses a local weather database

## Technologies
- Python

## Purpose
This project was built as part of my learning journey in programming.

## How to run

1. Run the program:

```bash
python main.py
```

2. Enter one of these cities:
- Buenos Aires
- Cordoba
- Rosario
- Mendoza
- La Plata
